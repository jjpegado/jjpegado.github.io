Projeto

O projeto "Filmaria" é uma aplicação web desenvolvida em React, focada em consumir APIs de plataformas de filmes para proporcionar uma experiência dinâmica e envolvente aos usuários. Utiliza dependências modernas como React-Router-DOM para navegação eficiente, Axios para realizar requisições HTTP de maneira simplificada, e React-Toastify para exibir notificações personalizáveis. Além disso, a biblioteca @Testing-Library assegura testes eficazes dos componentes, enquanto Web-Vitals monitora e otimiza a performance da aplicação. Essas ferramentas combinadas garantem uma interface de usuário fluida, responsiva e de fácil manutenção.
